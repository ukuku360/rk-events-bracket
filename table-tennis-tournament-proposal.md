# Roomingkos Swanston Table Tennis Tournament Proposal

## 1) Event Overview
- **Event type:** Table Tennis Tournament
- **Venue:** Roomingkos Swanston
- **Expected participants:** 16
- **Competition style:** Single-elimination bracket (knockout) with 3rd place playoff

## 2) Match Structure
- **Round of 16:** 8 games
- **Quarter-finals:** 4 games
- **Semi-finals:** 2 games
- **Final:** 1 game
- **3rd place playoff:** 1 game (to determine 3rd place)

**Total games:** `16` (including the 3rd place match)

## 3) Path to Prizes
- **1st place (champion):** 4 wins
- **2nd place (runner-up):** 3 wins, 1 loss
- **3rd place:** 3 wins, 1 loss in semi-final, 1 win in 3rd place match

## 4) Prize Plan (low-cost, Woolworths gift cards)
- **1st place:** Woolworths gift card – **$50**
- **2nd place:** Woolworths gift card – **$30**
- **3rd place:** Woolworths gift card – **$20**
- **Total prize cost:** **$100**

### Optional lower-cost alternative
- **1st place:** $40
- **2nd place:** $25
- **3rd place:** $15
- **Total:** $80
